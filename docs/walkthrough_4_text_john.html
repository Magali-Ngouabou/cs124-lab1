Renaming and Item

Before

Here, the page contains multiple items in the list, one of which is named "Text John". To begin renaming this item, the user will click on the small edit icon shaped like a pen on paper to the right of the item

Editing...

Now that the edit icon has been clicked, the item "Text John" has been replaced with a textbox with "Text John" already filled out. To make an edit to the name, the user can change the text to be the desired name, in this case, "Text John about bank statements"

After

The user will hit enter, and the change will be made to the item
