Showing Only Uncompleted Items

Before

Here, the page contains multiple items in the list, one of which is named "Call Mom" which is marked as completed from before. To show only the uncompleted items (hiding the completed items like "Call Mom"), the user will toggle the switch next to the "Hide Completed Tasks" header

After

The user toggles the switch, which moves to the right and turns green to signify that uncompleted items are being hidden
